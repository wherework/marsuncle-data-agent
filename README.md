**MarsUncle DataAgent（火星叔叔 DataAgent）** 是一个面向真实业务场景的 **结构化数据自然语言查询与分析智能体**，用于探索比传统 Text-to-SQL 更完整、更可产品化的“问数”方案。

---

## 项目缘起

在大量实际项目中我们发现，“问数”并不只是把一句话翻译成 SQL：

- 用户表达往往是模糊、不完整、甚至前后不一致的  
- 业务口径（指标、维度、时间、过滤条件）需要被正确理解和对齐  
- 查询结果需要可解释、可验证，而不仅仅是“能跑”  
- 系统必须具备可维护性，而不是一次性 Demo  

**MarsUncle DataAgent** 希望从 **产品化、工程化、可演进** 的角度，重新思考数据查询智能体的设计方式。

---

## 它是什么 / 它不是什么

### 它试图成为：

- 一个 **面向结构化数据的 Agent 架构**  
- 能够理解业务语义的查询与分析系统  
- 具备清晰领域模型（指标 / 维度 / 过滤条件 / 时间语义）  
- 可被真实系统集成，而不仅是实验性代码  

### 它不是：

- 纯粹的 Text-to-SQL 模型封装  
- 简单的 LLM 聊天示例  
- 只为展示效果、不考虑工程落地的 Demo
